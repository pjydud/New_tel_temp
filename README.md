# New_tel_temp

## 현재 알림 범위

이 저장소의 주식 알림은 매일 09:10 KST 기준으로 코스피·코스닥 전체 종목의 누적 거래량을 확인하고, 최근 7회 같은 시간대 평균 거래량 대비 급등률이 높은 5개 종목을 텔레그램으로 전송합니다.

## 제외된 검색 범위

아래 항목은 더 이상 검색하거나 알림에 포함하지 않습니다.

- 인물 기사 검색
- 화성시 기사 검색
- 더불어민주당 관련 기사 검색
- 기타 정치·지역 뉴스 요약

## 동작 방식

1. GitHub Actions가 매일 09:10 KST에 실행됩니다.
2. `pykrx`로 코스피·코스닥 거래량을 조회합니다.
3. `data/volume_snapshots.json`에 09:10 거래량 스냅샷을 저장합니다.
4. 최근 7회 동일 시간대 평균 대비 거래량 급등률 상위 5개 종목을 텔레그램으로 전송합니다.

## 참고

초기에는 최근 7회 비교 데이터가 부족할 수 있으므로, 며칠간은 스냅샷 저장 안내 메시지만 발송될 수 있습니다.
